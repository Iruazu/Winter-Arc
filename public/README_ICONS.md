# PWA Icons

このディレクトリには以下のPWAアイコンファイルを配置してください:

- `pwa-192x192.png` (192x192 pixels)
- `pwa-512x512.png` (512x512 pixels)

アイコンがない場合、PWA機能は動作しますが、ホーム画面追加時のアイコンが表示されません。

## 簡単なアイコン作成方法

1. Canva や Figma などで 512x512 のアイコンを作成
2. 背景色: #0f172a (midnight-900)
3. シンボル: 雪の結晶や山のシルエットなど
4. テキスト: "WA" (Winter Arc の略) など

または、以下のツールで簡単に作成できます:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
